# EscapeRoom

FPP Escape Room game made with Unreal Engine 5.5

# Introduction

Escape room is a project primarly made for class but it is evolving in my first portoflio entry.
There's bunch of mechanics:
- [Interact System](#interact-system),
- key/lock system,
- UV flashlight,
- interactive keypads,
- tv for movies and interactive widgets (hangman, RGB to HSV converter)
